# DEVLOG 2026-07-02 — cnu → jnu 리브랜딩 & 전남대 CI 적용

## 개요
전남대학교 사이트의 리포/도메인/식별자를 올바른 약자 **jnu**(전남대 공식 도메인 jnu.ac.kr 컨벤션)로 전면 변경하고, 전남대 공식 CI(심볼·시그니처·엠블럼)를 적용. 배포 과정에서 발생한 로그인 장애도 복구.

## 1. cnu → jnu 전면 리브랜딩
- 전 코드/설정/SQL/문서 일괄 치환: `cnu → jnu`, `CNU → JNU`
- 도메인: `cnu.dreamitbiz.com` → **`jnu.dreamitbiz.com`** (CNAME, gh-pages 모두)
- DB 접두사: `cnu_` → **`jnu_`** (`src/config/site.ts` dbPrefix)
- 컬러테마 id `cnu` → `jnu`, 브랜드 텍스트 `CNU` → `JNU`
- 영문 정식명 **Chonnam National University**는 유지
- GitHub 리포: `aebonlee/cnu` → **`aebonlee/jnu`** (원격 URL 변경), 로컬 폴더도 `jnu`로 변경

## 2. 전남대 공식 CI 로고 적용
- 원본 zip(심볼/시그니처/엠블럼)은 `brand/`에 보관(배포 제외), PNG 추출 후 `public/`에 배치
- 네비바 로고: **국영문 가로 시그니처** — 라이트용 `logo-jnu.png` + 다크모드용 `logo-jnu-dark.png`(심볼 초록 유지·글자만 흰색), `navbar.css`에서 `data-theme` 토글
- 파비콘: **전남대 심볼** 기반 PNG 세트(favicon-32 / apple-touch-icon / icon-192 / icon-512)

## 3. OG(Open Graph) 이미지
- `canvas` 패키지로 전남대 그린 브랜딩 OG 신규 생성: `scripts/generate-og.cjs` (`npm run og-image`)
- 시스템 폰트 AppleSDGothicNeo 사용, 1200×630
- **캐시 무력화**: 파일명을 `og-image.png` → **`og-jnu.png`**(새 URL)로 변경. og 메타 보강(`og:image:alt/type/secure_url`, `og:locale`, `twitter:image:alt`)
- 카카오 공유 디버거에서 캐시 초기화 후 재스크랩 → 새 초록 JNU 이미지 반영 확인

## 4. 인프라(DNS / GitHub Pages)
- DNS: `dreamitbiz.com`은 Cloudflare 관리, `*.dreamitbiz.com` 와일드카드 프록시로 **jnu 서브도메인 자동 연결**(별도 레코드 불필요)
- GitHub Pages: 커스텀 도메인 `jnu.dreamitbiz.com` 자동 등록(CNAME 기반), 소스 gh-pages 브랜치
- HTTPS: Cloudflare 엣지 인증서로 정상 동작(GitHub "Enforce HTTPS"는 프록시 구조상 미적용 — 기존 cnu와 동일)

## 5. 로그인 장애 복구 (중요)
- **증상**: 배포 후 "Supabase not configured"로 로그인 불가
- **원인**: `.env` 없이 클린 빌드하여 `VITE_SUPABASE_*` 값이 번들에서 누락(fallback 부재)
- **조치**: `src/utils/supabase.ts`에 공유 프로젝트(hcmgdztsgjvzcyxyayaj) URL/anon key **fallback 하드코딩**(daegu/data/hufs 등 형제 사이트와 동일 패턴)
- GitHub Pages 게시 지연 시 빌드 API 강제 재요청으로 전파 → 라이브 번들에 Supabase 설정 포함 확인, 로그인 복구
