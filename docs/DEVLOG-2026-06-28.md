# CNU 개발일지 — 2026-06-28

전남대학교 생성형 AI 교육 사이트(`cnu.dreamitbiz.com`)를 **전면 개편**했다.
기존 "교수자/직원/조교 3분류 + 상거래" 사이트를, **전남대GPT(타임리GPT 기반) 실습 중심의
4개 과정 학습 플랫폼**으로 재구성하고 디자인·내비게이션·콘텐츠를 새로 만들었다.

- 스택: React 19 · Vite 7 · TypeScript · Supabase · PortOne
- 배포: 수동 `npm run deploy`(gh-pages), 커스텀 도메인 `cnu.dreamitbiz.com`, DB 접두사 `cnu_`
- 위치: `~/dreamit-web/01-edu-sites/cnu`

---

## 1. 콘텐츠 구조 — 4개 과정 × 2일 16시간

3분류를 폐기하고 4개 전문 과정으로 재편. 교육과정·학습자료·커뮤니티가 **동일 id 체계**로 통일.

| id | 과정 | Day 1 | Day 2 |
|----|------|-------|-------|
| `foundation` | 생성형 AI 기초·심화 | 전남대GPT·프롬프트 기초 | 파일활용·맞춤GPT·자동화 |
| `research-writing` | 데이터분석 & 논문작성 | 데이터 정리·통계·시각화 | 문헌검토·IMRaD·영문교정·투고 |
| `data-budget` | 데이터분석 & 예산회계 | 엑셀·집계·시각화·검산 | 예산편성·정산·재무보고·규정Q&A |
| `research-lab` | 연구 & 실험지원 | 연구설계·계획서·IRB | 프로토콜·데이터기록·분석코드 |

- 데이터: `src/data/courses.ts`의 `PROGRAMS`(Day별 교시 커리큘럼 + `PracticeCase` 예시 프롬프트)
- 교시 = **50분 단위**(1·2교시 09:00~10:50 …), 하루 **8시간 · 4과목**
- 각 과정 마지막 교시에 **'평가하기'** 실습 포함
- 플랫폼은 **전남대GPT(타임리GPT 기반)** 기준, 실습 사례를 풍성하게(실습 위주)

## 2. 디자인 리뉴얼

- 4개 컬러셋 `ColorTheme = cnu | academic | research | vibe` (data-color 속성)
  - **기본 = `cnu` 전남대 다크 그린 #00553F** (ThemeContext 기본값·컬러피커 첫번째·theme-color)
  - 후보: academic(네이비#1A2B4A+골드#C8A04A) / research(틸) / vibe(바이올렛)
- serif 제목(Noto Serif KR), 본문 컨테이너 **1480px**(상단 메뉴 너비와 일치), html 폰트 **17px**
- 에디토리얼 히어로(좌 타이틀·CTA / 우 지표·일정카드) + 마퀴, chosun 스타일 정보 섹션
  (교육 특징 · 준비사항 · 교육 안내)
- 한국어 워드랩 방지: `word-break: keep-all` + `overflow-wrap: break-word` 전역

## 3. 내비게이션

상단: **4개 과정** + 프롬프트 학습 + 부록 + 추천사이트

- **AI도구 메뉴 삭제**(해당 없음) — 메뉴·푸터·홈 섹션 제거
- **프롬프트 학습**(`/prompt-eval`): 좌측 공통 `PromptEvalNav`(학습하기·실습하기·평가하기, 접고펴기)
- **부록**(`/appendix`): GuidePage로 용어 해설·실전 팁·함께 공부할 자료 제공
- 유틸 컨트롤(검색·언어·색상테마·화면모드·토큰충전)을 **집결**:
  로그인 시 개인(아바타) 메뉴 / 비로그인 시 '설정' 드롭다운, 이름 표기
- 과정 상세 **좌측 사이드바**: 커리큘럼 / 학습자료(드롭다운·인라인 표시) / 게시판 / 실습·따라하기 /
  Day 바로가기 / 다른 과정

## 4. 핵심 신규 기능

- **학습자료 인라인 표시**: 과정 사이드바에서 자료 클릭 시 페이지 이동 없이 우측 본문에 표시
  (`selectedMatId`, LearningMaterials는 `?m=<id>` 딥링크 지원)
- **실습 · 따라하기**: `src/data/handsOn.ts`의 `HANDS_ON[programId]` — 과정별 3~4개 랩,
  각 랩은 단계별 미션 + 전남대GPT 예시 프롬프트 + 확인 포인트 + 완료 결과
- **부록**: `src/pages/appendix/`(glossary · tips · resources)

## 5. 운영 정보 반영

- **별도 수강신청 없음** — 학교 모집·배정. '신청' 표현 제거, 버튼은 '과정 살펴보기'·'로그인하고 학습 시작'
- 홈 안내 카드: 담당 교수 **이애본·정동엽** / 자료열람 회원가입·로그인 후 / 문의 **aebon@hs.ac.kr**
- 추천사이트 깨진 아이콘 수정(Gemini→`fa-gem`, Copilot→`fa-microchip`), 라이브 UI 이모지 정리
