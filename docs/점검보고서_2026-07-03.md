## 3. 발견·수정한 결함 (주요)

1. **로그인 장애("Supabase not configured")**: .env 없는 클린 빌드로 설정 누락 → `supabase.ts` fallback 하드코딩으로 복구.
2. **다크모드 흰배경**: `html[data-color="jnu"]`가 특이도로 다크 규칙을 이겨 `--bg-light-gray` 등이 밝게 유지 → `[data-theme="dark"][data-color="jnu"]`에 다크 중립색 추가(근본 수정). 히어로 제목 대비 1.1→14로 개선.
3. **학습자료 불일치**: 데이터분석&예산회계/연구실험지원에 타 과정(행정) 자료가 잔존 → 커리큘럼에 맞게 교체.
4. **버튼 아이콘 간격**: `.btn` gap 부재로 아이콘·텍스트 밀착 → gap:8px 적용.
5. **JS 문자열 종료 충돌**: 프롬프트 내 작은따옴표가 문자열을 조기 종료 → 전부 큰따옴표로 교정.

## 4. 남은 확인 필요 (사용자 조치)

- **소셜 로그인 리다이렉트 실사용 1회 검증**: 도메인 변경(cnu→jnu)에 따라 Supabase Auth의 Redirect URL 허용목록에 `https://jnu.dreamitbiz.com/**` 포함 여부. 미허용 시 Dashboard → Authentication → URL Configuration에서 추가.
- **구 cnu 도메인**: 현재 404(jnu로 이전). 필요 시 Cloudflare 301 리다이렉트.
