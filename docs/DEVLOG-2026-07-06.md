# DEVLOG 2026-07-06 — jnu→cnu 재전환 · 복사블록 통일 · 부록 메뉴 확장

전남대학교 생성형 AI 교육 사이트(리포 `aebonlee/cnu`)에서 진행한 하루치 작업 기록.

## 1. jnu → cnu 재전환 (도메인·OG·타이틀·패들렛)

GitHub 레포는 이미 `aebonlee/jnu` → `aebonlee/cnu`로 rename된 상태였고(로컬 remote만 옛 주소로 남아 있어 정정), 코드/도메인/브랜딩을 cnu로 재정렬.

- **OG 이미지·태그**: `og-jnu.png` → `og-cnu.png`(중복 `og-image.png` 제거), `scripts/generate-og.cjs` 출력 경로·푸터 텍스트도 cnu로 변경 후 이미지 재생성. `index.html`의 og:image / twitter:image 참조 갱신.
- **도메인**: `jnu.dreamitbiz.com` → `cnu.dreamitbiz.com` (CNAME 2곳, `index.html` canonical·og:url·이미지 5곳, `src/config/site.ts` url, generate-og 푸터). DNS는 Cloudflare 와일드카드로 cnu 서브도메인 이미 해석됨.
- **타이틀**: `JNU AI Education` → `CNU AI Education` (`index.html <title>`, `site.ts` name, `translations.ts` title/subtitle).
- **패들렛 링크 6종**: `padlet.com/aebon/jnu2601~2606` → `cnu2601~2606` (`src/data/courses.ts` 5곳, `RecommendedSites.tsx` 1곳). ※ 패들렛 보드 slug 자체도 cnu26XX로 맞춰야 링크 유효.
- DB 접두사 `jnu_`, 내부 테마 키·로고 파일명은 별도 마이그레이션 사안이라 유지.

## 3. 복사 블록 디자인 전면 통일

사이트 전체 복사 UI를 하나의 디자인 언어로 통일.

- **복사 구현은 2종뿐**: `PromptBlock`(프롬프트/실습), `CodeBlock`(마크다운 코드펜스). 둘 다 **남색 헤더형**(좌측 라벨 + 우측 복사 버튼 + 라운드 래퍼)으로 통일.
- **단색 평문화**: CodeBlock의 무지개색 구문강조 제거 → 프롬프트블록과 동일한 단색(`#E8EEF8`) 평문(주석만 은은한 파랑).
- **이중 박스 제거(2단계)**:
  1. `.markdown-body pre/code`가 CodeBlock 안쪽 pre/code에 이중 적용되던 것 리셋(`.code-block-wrapper` 스코프 상향).
  2. 근본 원인 — `markdownComponents`가 `code`만 오버라이드하고 `pre`는 기본값이라 ReactMarkdown이 CodeBlock을 기본 `<pre>`(`.markdown-body pre` = `#1e1e2e` 검은 박스)로 **이중 래핑**. `pre` 언래핑(`pre: () => <>{children}</>`)으로 바깥 검은 박스 제거.
- **전 마크다운 경로 통일**: `StreamingOutput`(AI 답변)·`BoardDetail`(게시판)이 `markdownComponents`를 안 넘기던 것 연결. `GuidePage`의 **로컬 markdownComponents**에도 `pre` 언래핑 추가(부록·프롬프트학습 재발 방지).
- **eval-prompt-box**(프롬프트 평가 퀴즈 박스, 복사기능 없음)도 프롬프트블록 본문 톤(남색 `#14233E`·크림 텍스트·모노스페이스)으로 정렬.

## 4. 부록 메뉴 3종 추가

`src/pages/appendix/data/`에 데이터 파일 추가 후 `Appendix.tsx` `dataFiles`에 연결(각 파일 = 사이드바 메뉴 그룹, `sections` = 하위 항목).

- **클로드 활용 100가지**(`claude100.ts`): 7개 카테고리(프롬프트 기본기 / 글쓰기·콘텐츠 / 리서치·분석 / 업무·비즈니스 / 코딩·자동화 / 학습·정리 / 고급 테크닉).
- **ChatGPT 비밀코드**(`chatgptCodes.ts`): 11개 카테고리 코드표(핵심 5선·글쓰기·설명난이도·형식·학습·아이디어·분석·고급사고·창작·코딩·파워명령어).
- **ChatGPT 실전 사용법**(`chatgptGuide.ts`): 실전 핵심 요약 + 슬래시 비밀코드 100개(20개씩 5섹션 표).
- 모두 KO/EN 이중 콘텐츠.

## 5. 배포 — GitHub Pages 레이트리밋 이슈

- 배포 방식: 수동 `npm run deploy`(gh-pages 브랜치) → `pages-build-deployment` 액션.
- **하루에 배포를 짧은 간격으로 다수 실행**해 GitHub Pages가 중복·연속 배포를 레이트리밋 → "Deploy to GitHub Pages" 단계에서 `Deployment failed, try again later`, 재실행이 "re-run pending"으로 교착.
- 빌드 산출물·gh-pages 브랜치 내용은 정상. **대응: 이후 커밋·푸시·배포는 2~3분 간격**을 두고 진행할 것.
