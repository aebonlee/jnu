## 1. 부록(Appendix) 대개편

### 1-1. 콘텐츠 정리·큐레이션
- **클로드 활용 100가지**: 100개 팁 각각에 **실사례 예시 프롬프트** 삽입, 제목·설명 분리 카드형(#### 헤딩), 이모지·중복번호 제거.
- **AI 키워드 메뉴 통합**: 'ChatGPT 비밀코드'→'ChatGPT 활용 키워드'→**'AI 활용 키워드'**로 정정(비공식 명령을 오인 방지, 클로드 등 공통). 중복되던 두 키워드 메뉴를 **'AI 실전 사용법' 하나로 통합**하고 *정말 잘 통하는 핵심만* 큐레이션(형광펜 표시).
- 임의 코드(SELFREFINE 등)는 "구분용 별칭(키워드)"임을 명시, 슬래시(/) 유무는 기능 아닌 표기 습관임을 안내.
- 약어 **원어 병기**: ELI5·TL;DR·SWOT·AHP 등. **페르소나** 표기 통일(국립국어원). **용어 해설에서 '전남대GPT' 항목 삭제**.

### 1-2. AI 실전 사용법 (직원·조교 실습 지향)
- 실전 핵심 요약(사무 맥락) · **자주 쓰는 핵심 키워드**(형광펜+복사 예시) · **설명 수준·대상 정하기**(초등~전문가 6단계 난이도 사다리, AHP 예시로 아카데믹) · **톤·스타일 옵션** · **한국어 특성**(존댓말·번역투·개조식·글자수) · **자주 겪는 문제**(한글 인코딩·워드 문서 기호 제거).

### 1-3. AI 모델 고르기 (신설)
- 기본 프리셋(회사별 대표 1종+Auto) · 등급 3가지 · 목적별 추천 · **모델 선택 기준(전문 관점 6축·유형)** · **제공사별 상세**(Upstage·OpenAI·Google·Anthropic·Meta·Mistral·Qwen·Grok — 프로필+유형/강점/용도/비고).
- 벤치마크 수치는 지어내지 않고 **아키텍처·용도 중심** + "잘 모르면 Auto" 원칙. 제공사별 섹션은 **콘텐츠 제목에만 '(2026.07.07 기준)'** 표기(사이드바는 날짜 없음).

## 2. 다크모드·모바일 전면 대응

부록 신규 CSS의 하드코딩 밝은색(`#E8F5E9`·`#1B5E20`·`#1e1e2e`·`#C8E6C9` 등)이 다크에서 깨지던 문제를 전수 수정.
- **테마 변수·반투명 틴트로 교체**: 사이드바 박스 그룹, 콘텐츠 카드, 표(헤더 밝은 녹색/셀 크림), h3 헤딩(밝은 그라데이션→반투명), TipBox(틴트+text-primary), 형광펜(mark).
- **TipBox 다중 볼드 문장깨짐 수정**: `strong{display:block}`이 한 문장 속 여러 볼드를 줄바꿈시키던 것 → **맨 앞 라벨만 블록, 중간 볼드는 인라인**.
- **반응형 미디어쿼리** 추가(≤1024/≤768): 사이드바 세로 스택·전체폭.
- **전 26개 섹션 다크모드 자동 검사**: 밝은 배경·저대비 텍스트 0건 확인.

## 4. 실습 워크북 신설 (4개 과정 전체)

과정 페이지(`CourseCategory`) 사이드바에 **'실습 워크북' 드롭다운**을 학습자료처럼 추가하고, **기존 '실습·따라하기'를 그 하위로 편입**.
- 데이터: `src/data/workbook.ts` — `WORKBOOK[programId] = WorkbookModule[]`, 각 모듈 = markdown(단계별 복사용 프롬프트는 코드펜스 → 복사 가능한 CodeBlock).
- 뷰: CourseCategory에 `selectedWbId` 분기 추가(`.material-inline` + `markdownComponents`로 렌더).
- 초기 13개 모듈(기본/응용/심화)로 시작 → **학습자료와 중복** 지적을 반영해 재구성.

### 4-2. 최종 구조 — **둘째날 8시간을 워크북만으로** (이원 트랙, 4과정 × 5모듈 = 20)
데이터분석이 主·주제(논문/예산/연구)가 補인 실제 수강 상황을 반영해, 대상별로 갈라지는 이원 트랙으로 확대.
- 각 과정: **① 오늘의 실습 안내(8시간 시간표·트랙 선택·시니어 팁) + ②③ [공통](데이터·AI 기본·응용) + ④ [트랙 A](주제 심화: 논문·예산·연구) + ⑤ [트랙 B](업무 활용: 비전공·다수)**.
- 학습자료를 '기반'으로 참조(반복 아님), 손으로 재현→내 업무 적용→통합. 시니어 초보자 접근성(복사 3단계·"막히면"), 전남대GPT 실전, 업무 적용 중심.
- 논문작성(5명)은 트랙 A(ConnectedPapers·FINER·IMRaD·심사비평), 관련 덜한 다수는 트랙 B(내 데이터 보고·자동화·템플릿)로 모두 충족.
